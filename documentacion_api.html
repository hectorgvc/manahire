Manahire API

Guía de integración para portales web corporativos

1. Introducción

La API pública de Manahire permite que sitios web externos o el portal principal de la empresa consuman la lista de vacantes activas en tiempo real. Este endpoint está diseñado para ser seguro, filtrando automáticamente cualquier dato sensible de candidatos o notas internas de reclutamiento.

2. Endpoint de Vacantes

Utiliza este recurso para obtener todas las posiciones marcadas como Abierta.

GET Obtener Vacantes

https://manahire.vercel.app/api/public/jobs

Parámetros: No requiere autenticación (Solo lectura pública).

3. Ejemplo de Integración (JavaScript)

Puedes copiar este código en el sitio web donde desees mostrar las vacantes:

async function cargarVacantes() {
    const URL_API = 'https://manahire-portal.vercel.app/api/public/jobs';
    
    try {
        const respuesta = await fetch(URL_API);
        const { success, data } = await respuesta.json();

        if (success) {
            console.log('Vacantes encontradas:', data);
            
            // Ejemplo de cómo dibujarlo en el HTML
            const contenedor = document.getElementById('lista-empleos');
            contenedor.innerHTML = data.map(job => `
                <div class="job-card">
                    <h3>${job.title}</h3>
                    <p>${job.department} - ${job.modality}</p>
                    <a href="/postular/${job.id}">Ver Detalles</a>
                </div>
            `).join('');
        }
    } catch (error) {
        console.error('Error al conectar con Manahire:', error);
    }
}

4. Estructura de la Respuesta

La API devuelve un objeto JSON con la siguiente forma:

{
  "success": true,
  "data": [
    {
      "id": "uuid",
      "title": "Senior Developer",
      "department": "Tecnología",
      "type": "Tiempo completo",
      "modality": "Remoto",
      "description": "...",
      "requirements": "...",
      "salary_range": "$2k - $3k",
      "created_at": "2024-02-26T..."
    }
  ]
}

5. Notas de Seguridad

  • Filtro de Estado: Solo se devuelven vacantes con status: 'Abierta'.
  • Privacidad: No se exponen datos de la tabla candidates ni campos de seguimiento interno.
  • CORS: Si el dominio de tu sitio web es bloqueado, solicita al administrador de Manahire añadir tu URL a la lista de permitidos en next.config.ts.
